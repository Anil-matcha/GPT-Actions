### Run the project locally

Minimum requirements to run the projects locally

- Node.js v18
- OpenAI API Key
- Python3

```shell

## Client

cd client

npm install

npm run build

npm start

## Server

cd ../server

pip install -r requirements.txt

python webserver.py
```

Use ngrok to test on a https endpoint
